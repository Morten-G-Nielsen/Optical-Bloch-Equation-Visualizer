# Bloch-Equation-Visualizer

A real time simulation of the optical Bloch equation in C++ using OpenGL.

## Keybinds
- Left rotates left
- Right rotates right
- Up rotates upwards
- Down reotates downwards
- 1 sets off a pulse
- 2 increses pulse width
- 3 decreses pulse width
- 4 inceases pulse area
- 5 decreases pulse area
- g selects gauss pulse
- s selects a square pulse
- c selects a complex pulse

## Requirements
- C++23 compiler
- OpenGL 3.3+
- GLFW
- Glad
- GLM
- make

## Build (Linux)
git clone https://github.com/Morten-G-Nielsen/Optical-Bloch-Equation-Visualizer.git

cd project

make
